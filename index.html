Open communications platform & API for the Internet of Things.

Mesh Network

Skynet is a cloud-based mesh network that scales to meet any needs whether the nodes are smart home devices, sensors, cloud resources, drones, bots, etc.

REST & WebSocket API

Skynet is powered by Node.JS and based entirely on its integrated REST API and realtime Websocket API with extensible JSON device & messaging docs.

Node.JS NPM Module

Node.JS is known for fast, event-driven operations perfect for nodes and devices such as RaspberryPi, Arduino, and Tessel. Our NPM module gets your app up & running quickly!

REST API

Status

/status
Returns the Skynet platform status
curl -X GET http://skynet.com/status
{"skynet":"online","timestamp":1381536589633,"eventCode":200,"_id":"5258934d56330f7dd0000009"}

Devices

/devices?key=value&key=value
Returns an array of device UUIDs based on key/value query criteria
curl -X GET http://skynet.com/devices?type=drone
{"devices":["ad698900-2546-11e3-87fb-c560cb0ca47b","2f3113d0-2796-11e3-95ef-e3081976e170"]}b
/devices/{uuid}
Returns all information (except the token) of a specific device or node
curl -X GET http://skynet.com/devices/ad698900-2546-11e3-87fb-c560cb0ca47b
{"_id":"5241d9140345450000000001","api":"update","armed":true,"channel":"main","deviceDescription":"this is a test","deviceName":"hackboard","key":"777","online":true,"socketId":"fO8dxKEwARUH9Ea--p2C","timestamp":1381536277933,"uuid":"ad698900-2546-11e3-87fb-c560cb0ca47b"}
/devices
Payload: key=value (i.e. type=drone&color=black)
Registers a node or device with Skynet. Skynet returns a UUID device id and security token. You can pass any key/value pairs and even override Skynet's token by passing your own token in the payload i.e. token=123
curl -X POST -d "type=drone&color=black" http://skynet.com/devices
{"type":"drone","color":"black","uuid":"6f72bc60-32d5-11e3-92e2-e94f95fc16a6","timestamp":1381537803046,"token":"4bbd2jm242dl5wmimbwz4rvlu77m0a4i","channel":"main","online":false,"_id":"5258980b56330f7dd000000d"}
/devices/{uuid}
Payload: token=123&key=value (i.e. token=123&type=drone&color=blue)
Updates a node or device currently registered with Skynet. You must pass the token for security but you can pass any key/value pairs to update object as well as null to remove a propery (i.e. color=null).
curl -X PUT -d "token=123&color=blue" http://skynet.com/devices/ad698900-2546-11e3-87fb-c560cb0ca47b
{"uuid":"ad698900-2546-11e3-87fb-c560cb0ca47b","color":"blue","timestamp":1381537750462}
/devices/{uuid}
Payload: token=123
Deletes or unregisters a node or device currently registered with Skynet. You must pass the token for security .
curl -X DELETE -d "token=123" http://skynet.com/devices/ad698900-2546-11e3-87fb-c560cb0ca47b
{"uuid":"ad698900-2546-11e3-87fb-c560cb0ca47b","timestamp":1381537971074}

Messages

/messages
Send a message to a specific device, array of devices, or all devices on the Skynet platform
curl -X POST -d '{"devices": "all", "message": {"yellow":"off"}}' http://skynet.com/messages
curl -X POST -d '{"devices": ["ad698900-2546-11e3-87fb-c560cb0ca47b","2f3113d0-2796-11e3-95ef-e3081976e170"], "message": {"yellow":"off"}}' http://skynet.com/messages
curl -X POST -d '{"devices": "ad698900-2546-11e3-87fb-c560cb0ca47b", "message": {"yellow":"off"}}' http://skynet.com/messages
{"skynet":"online","timestamp":1381536589633,"eventCode":200,"_id":"5258934d56330f7dd0000009"}
{"devices":"all","message":{"yellow":"off"}
